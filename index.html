siplib.io

a modern and powerful library for parsing & manipulating SIP messages

SIP LIb is a powerful SIP library for framing & parsing SIP messages with a fluent API and immutable objects. SIP Lib is part of pkts.io and is designed to be used as a standalone module, either for building your own SIP stack from scratch, or developing your own tools for parsing & manipulating SIP.
